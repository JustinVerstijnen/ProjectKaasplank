## Introduction to the CWL Active Directory Red Team Specialist

After gaining the Azure Read Team Specialist certification, I wanted to further strengthen my knowledge into hybrid environments. I therefore did the Cyberwarfare.live Active Directory Red Team Specialist course.

## 1: Core components of Active Directory

The core components of the Active Directory service are:

- Domain Controllers: Server running AD DS and processing authentication requests
- Schema: All object types and their attributes
- Global Catalog: Subset of AD attributes for fast and forest-wide searches
- NTDS.dit: This is the Active Directory database file, shared among multiple domain controllers
